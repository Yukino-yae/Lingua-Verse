# Lingua-Verse

> Learning every Language you want.

A comprehensive language learning platform with interactive features for vocabulary management, listening practice, and more.

## Features

- **Language Learning** - Vocabulary learning with Ebbinghaus memory curve for optimal retention
- **Listening Practice** - Audio/video player with synchronized subtitles, word lookup, and vocabulary collection
- **Vocabulary Management** - Built-in word banks for English (CET-4, CET-6, IELTS/TOEFL), Japanese (N5-N1), and more
- **Multi-language Support** - English, Japanese, Korean, French, German, Spanish, Italian, Russian, and Classical Chinese
- **Theme Toggle** - Dark/Light mode support
- **Responsive Design** - Works on desktop and mobile devices

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage with feature overview |

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/your-username/Lingua-Verse.git
```

2. Open with a local server
```bash
npx http-server -p 8080
```

3. Visit `http://localhost:8080` in your browser

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Web Speech API (Text-to-Speech, Speech Recognition)
- LocalStorage for data persistence
